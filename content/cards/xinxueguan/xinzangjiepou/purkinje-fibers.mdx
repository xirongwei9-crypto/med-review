## 位置与分布

- 位于**心室内膜下**（心内膜深层）
- 由左、右束支分支延续形成
- 在心室内膜下形成**浦肯野纤维网**，遍布两个心室

## 组织学特征

| 特征 | 浦肯野纤维 | 普通心肌细胞 |
|------|-----------|-------------|
| **直径** | 粗大（35-40μm） | 较细（10-15μm） |
| **肌原纤维** | 少，排列稀疏 | 丰富，排列紧密 |
| **横纹** | 不明显 | 明显 |
| **糖原** | 丰富 | 较少 |
| **闰盘** | 丰富，间隙连接多 | 一般 |
| **传导速度** | **极快**（2-4 m/s） | 较慢（0.4-0.5 m/s） |

> 浦肯野纤维是心脏中**传导速度最快**的组织

## 功能

### 快速传导

- 将房室束传来的冲动**迅速**传导到整个心室肌
- 传导速度是普通心室肌的 **5-10倍**

### 同步心室收缩

- 左右心室的心内膜下逐层传导 → 心内膜先收缩 → 心外膜后收缩
- 保证心室肌细胞**同步性收缩** → 有效射血

### 三级起搏点（备用）

- 自律性 15-40 次/分
- 当窦房结和房室结均失效时，浦肯野纤维可"接替"成为起搏点
- 但频率较慢 → 称为**室性逸搏心律**

## 传导通路总结

```
窦房结 → 结间束 → 房室结 → 房室束(His束) → 左右束支 → 浦肯野纤维网 → 心室肌
         ~50ms      ~100ms     ~30ms          ~20ms       ~30ms（全室激动）
```

**总传导时间**：窦房结 → 全室激动完成约 **220ms**

## 临床联系

### 束支传导阻滞

| 类型 | 特点 |
|------|------|
| **右束支传导阻滞 (RBBB)** | 右室激动延迟；V1导联 RSR' 型（M型）；较常见，可见于正常人 |
| **左束支传导阻滞 (LBBB)** | 左室激动延迟；V5-V6导联宽大R波；通常提示器质性心脏病 |

### 早后除极 (EAD) 与触发活动

- 浦肯野纤维的动作电位时程较长
- 在某些条件下（低钾、药物、心动过缓）易发生**早后除极**
- 可触发**尖端扭转型室速**（Torsade de Pointes）

### 室性心律失常

- 浦肯野纤维系统是**室性早搏（PVC）**和**室性心动过速**的常见起源
- 心肌缺血时浦肯野纤维自律性增高 → 室性心律失常
