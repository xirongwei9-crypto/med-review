## 右心房 Right Atrium

### 结构特点
- **三个入口、一个出口**
- 入口：**上腔静脉口**、**下腔静脉口**、**冠状窦口**
- 出口：**右房室口**（通向右心室）

### 重要结构
| 结构 | 说明 |
|------|------|
| **界嵴/界沟** | 分界：腔静脉窦 vs 固有心房 |
| **腔静脉窦** | 内壁平滑，上下腔静脉和冠状窦开口于此 |
| **固有心房** | 包括**右心耳**，内壁有梳状肌，不平滑，**易形成血栓** |
| **卵圆窝** | 房间隔右侧面的凹陷，是胎儿卵圆孔的遗迹 |
| **下腔静脉瓣** | 引导胎儿期血液流向卵圆孔 |
| **冠状窦口** | 心静脉血回流入口 |

## 右心室 Right Ventricle

### 结构特点
- 分**流入道（窦部）**和**流出道（动脉圆锥/漏斗部）**
- 两者以**室上嵴**为界

### 三尖瓣复合体 Tricuspid Valve Complex
| 组成 | 功能 |
|------|------|
| **三尖瓣环** | 纤维环，瓣膜附着处 |
| **三尖瓣**（前瓣、后瓣、隔瓣） | 防止血液从右心室逆流回右心房 |
| **腱索** | 连接瓣膜与乳头肌 |
| **乳头肌** | 收缩时牵拉腱索，防止瓣膜翻转 |

### 特殊结构
- **隔缘肉柱**（节制索）：内含右束支，连接室间隔与前乳头肌
- **肺动脉瓣**：位于流出道末端，防止血液逆流回右心室

## 记忆口诀

> 右房三入一出：上腔、下腔、冠状窦 → 右房室口
>
> 右室："室上嵴分流入流出，三尖瓣+腱索+乳头肌=三尖瓣复合体"
