## 冠脉循环路径

```
升主动脉 → 左、右冠状动脉 → 分支 → 毛细血管网 → 各级心静脉 → 冠状窦/心前静脉 → 右心房
```

## 心静脉系统

### 三大回流途径

| 途径 | 静脉 | 回流至 |
|------|------|--------|
| **冠状窦途径（主要）** | 心大静脉 + 心中静脉 + 心小静脉 | 右心房 |
| **心前静脉** | 右室前面的小静脉 | 直接开口于右心房 |
| **心最小静脉**（Thebesian veins） | 心壁内的小静脉 | 直接开口于各心腔 |

### 冠状窦 Coronary Sinus
| 属支 | 走行 |
|------|------|
| **心大静脉** | 与前室间支伴行 → 冠状沟左侧 → 冠状窦 |
| **心中静脉** | 与后室间支伴行 → 冠状窦 |
| **心小静脉** | 右缘 → 冠状窦 |

### 重要概念
- 冠状窦口开口于**右心房**（位于下腔静脉口与右房室口之间）
- 左右冠状动脉之间有**广泛吻合**，但吻合支**细小**

## 临床联系

- **急性心梗**时侧支循环不能很快建立（吻合支太细小）→ 心肌缺血坏死
- **冠脉造影**评估狭窄程度 → 决定是否需要支架/搭桥
- 心肌梗死心电图：**ST段抬高**（弓背向上），与直立T波连接
- **心肌酶谱**升高：肌钙蛋白、CK-MB

## 记忆口诀

> "三大回心路：冠状窦为主，心前直接入，最小散在各腔"
>
> 冠脉循环：升主→冠脉→分支→毛细血管→心静脉→冠状窦→右房
