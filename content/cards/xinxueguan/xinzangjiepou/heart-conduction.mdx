## 传导通路（6级）

```
窦房结 → 结间束 → 房室结 → 房室束（His束）→ 左右束支 → 浦肯野纤维网
```

## 各结构详解

| 结构 | 位置 | 功能 |
|------|------|------|
| **窦房结** | 上腔静脉与右心房交界处 | **正常起搏点**，自主节律最高（60-100次/分） |
| **结间束** | 心房内 | 连接窦房结与房室结 |
| **房室结** | 右心房下部、冠状窦口前方 | **传导延迟**，保证心房先收缩、心室后收缩 |
| **房室束**（His束） | 室间隔膜部 | 唯一连接心房与心室的电通道 |
| **左右束支** | 室间隔两侧 | 传导到左右心室 |
| **浦肯野纤维网** | 心室内膜下 | 快速传导，同步心室收缩 |

## 关键概念

- 心纤维骨骼**绝缘**心房与心室肌 → 房室束是**唯一**电通道
- 房室结传导**延迟**约0.1秒 → 保证心房收缩完成后再开始心室收缩
- 窦房结自律性最高 → 主导整个心脏节律（**窦性心律**）

## 临床联系

- **病态窦房结综合征**（SSS）：窦房结功能减退 → 心率过慢
- **房室传导阻滞**：I度/II度/III度
- **预激综合征**（WPW综合征）：存在异常旁路（Kent束），绕过房室结
- **室上嵴**：右心室内结构，内含右束支（节制索/隔缘肉柱）
