## 核心结构

革兰阳性菌（G+）细胞壁由**厚层肽聚糖**（15-50层）和**磷壁酸**组成。

### 主要成分

1. **肽聚糖层**：15-50层，占细胞壁干重的50%-80%
2. **磷壁酸**（Teichoic acid）：
   - **壁磷壁酸**：锚定在肽聚糖上
   - **膜磷壁酸**（LTA）：锚定在细胞膜上
   - 功能：抗原性、维持表面电荷、噬菌体受体

### 关键特点

- 无外膜（区别于G-菌）
- 对溶菌酶敏感
- 革兰染色呈**紫色**（结晶紫-碘复合物被厚肽聚糖层保留）
- 青霉素等β-内酰胺类抗生素的作用靶点

## 临床联系

- 磷壁酸与宿主细胞黏附有关（致病因子）
- LTA可激活TLR2 → 炎症反应
- 万古霉素通过与D-Ala-D-Ala结合阻断肽聚糖交联

## 记忆口诀

> "羊（阳）皮（肽）厚，有壁（磷壁）酸"
> 羊 = 阳性菌，皮 = 肽聚糖，厚 = 层数多，壁酸 = 磷壁酸

## 对比关键

| 特征 | G+ 菌 | G- 菌 |
|------|-------|-------|
| 肽聚糖厚度 | 厚 (15-50层) | 薄 (1-2层) |
| 外膜 | 无 | 有 |
| 磷壁酸 | 有 | 无 |
| LPS/内毒素 | 无 | 有 |
| 溶菌酶敏感性 | 高 | 低 |
